## Project Description
This is a Weather App that allows users to search for any city and view real-time weather information using the OpenWeatherMap API. It shows temperature, weather conditions, wind speed, humidity, and a simple forecast.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeatherMap API

---

## Features
- Search weather by city name
- Displays temperature in °C and °F
- Shows weather description and icon
- Displays humidity and wind speed
- Weather forecast preview
- Search history using local storage
- AI weather advice messages

---

## How to Run
1. Download or clone this repository
2. Open the folder in VS Code
3. Open `index.html`
4. Run using Live Server
